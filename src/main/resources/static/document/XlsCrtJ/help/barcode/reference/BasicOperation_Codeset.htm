バーコードクラスでは、CODE128、GS1-128 での FNC1 コードの挿入やコードセット切り替えには次の文字列を使用します。

キャラクタ文字列
CODE A (コードセット A)CODEA
CODE B (コードセット B)CODEB
CODE C (コードセット C)CODEC
FNC1FNC1

CODE128、GS1-128 で FNC1 コードの挿入やコードセットを切り替える方法

バーコードデータに ~ (チルダ) を入れて表の文字列を設定します。

コピーサンプルコード
// BarCodeCreator のインスタンスを生成します。
BarCodeCreator barCode = new BarCodeCreator();
// バーコードの種類を「Code128」に設定します。
barCode.setType(BarCodeType.Code128);
// バーコード生成単位を「Pixel」に設定します。
barCode.setUnit(Unit.Pixel);
// バーコードの横方向の解像度を「600」に設定します。
barCode.setDpiX(600);
// バーコードの縦方向の解像度を「600」に設定します。
barCode.setDpiY(600);
// バーコードの最小エレメント値を「6」に設定します。
barCode.setElement(6);
// バーコードデータの途中で FNC1 コードを設定し、更にコードセット C に切り替えます。
barCode.setValue("ABC~FNC1~CODEC1234567890");
// バーコードイメージを保存します。
try {
  barCode.saveImage("c:\\barcode.png", "PNG");
} catch (IOException e) {
  e.printStackTrace();
}
