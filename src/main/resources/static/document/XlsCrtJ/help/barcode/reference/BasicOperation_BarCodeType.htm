BarCodeCreator for Java
作成可能なバーコードの種類
参照

バーコードクラスでは、JAN、GS1-128、カスタマバーコード、QR コードなどのバーコードイメージを作成することが可能です。作成したバーコードイメージは、帳票ドキュメントに挿入してバーコード付きの帳票として出力することができます。バーコードクラスの対応バーコードを次に示します。

このトピックは、次のセクションを含みます。

  • JAN コード
  • UPC コード
  • ITF コード
  • CODABAR (NW-7)
  • CODE39
  • CODE128
  • GS1-128 (旧称 : UCC/EAN128)
  • 公共料金等代理収納 GS1-128
  • 郵便カスタマバーコード
  • QR コード
  • PDF417

バーコードの作成方法

JAN コード

JAN(13桁)
JAN(8桁)

コピーサンプルコード
// BarCodeCreator のインスタンスを生成します。
BarCodeCreator barCode = new BarCodeCreator();
// バーコードの種類を設定します。
barCode.setType(BarCodeType.JAN13);
// バーコード生成単位を設定します。
// setUnit メソッドに Pixel を指定し、setDpiX、setDpiY メソッドにデバイスの解像度を
// 指定することで高精度なイメージを作成することができます。
barCode.setUnit(Unit.Pixel);
// デバイスの解像度を設定します。
barCode.setDpiX(600);
barCode.setDpiY(600);
// バーコードの倍率を設定します。
barCode.setZoom(100);
// バーコード文字列を設定します。
barCode.setValue("491234567890");
// バーコードイメージを保存します。
try {
    barCode.saveImage("c:\\barcode.png", "PNG");
} catch (IOException e) {
    e.printStackTrace();
}

PDF417

標準
簡易

コピーサンプルコード
// BarCodeCreator のインスタンスを生成します。
BarCodeCreator barCode = new BarCodeCreator();
// バーコードの種類を設定します。
barCode.setType(BarCodeType.PDF417);
// バーコード生成単位を設定します。
// setUnit メソッドに Pixel を指定し、setDpiX、setDpiY メソッドにデバイスの解像度を
// 指定することで高精度なイメージを作成することができます。
barCode.setUnit(Unit.Pixel);
// デバイスの解像度を設定します。
barCode.setDpiX(600);
barCode.setDpiY(600);
// バーコードの最小エレメント値を設定します。
barCode.setElement(2);
// PDF モデルを設定します。
barCode.setPDFModel(PDFModel.Default);
// PDF の 1 モジュールの高さを設定します。
barCode.setPDFRatio(7.5);
// PDF の列数を設定します。(0 は自動設定)
barCode.setPDFColumn(0);
// PDF の行数を設定します。(0 は自動設定)
barCode.setPDFRow(0);
// PDF のエラー訂正レベルを設定します。
barCode.setPDFLevel(0);
// バーコード文字列を設定します。
barCode.setValue("0123456789");
// バーコードイメージを保存します。
try {
    barCode.saveImage("c:\\barcode.png", "PNG");
} catch (IOException e) {
    e.printStackTrace();
}
