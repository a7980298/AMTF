公共料金等代理収納 GS1-128 の設定について
参照

バーコードクラスの対応バーコードである「公共料金代理収納 GS1-128」では、内部で次に示す設定を行いバーコードイメージを作成します。

  • スタートコードセットを「コードセット C」に自動で設定します。
  • バーコードシンボル下の目視文字 (バーコードデータ) は、2 段で表示します。
  • 目視文字のみで表示する識別文字の括弧や指定位置に入れるハイフンは、自動で設定します。

「公共料金代理収納 GS1-128」 のバーコードを作成するには

次のバーコードイメージを作成するコードです。

コピーサンプルコード
// BarCodeCreator のインスタンスを生成します。
BarCodeCreator barCode = new BarCodeCreator();

String barCodeData;
barCodeData = "91";				// 識別子
barCodeData += "912345";			// メーカーコード
barCodeData += "1234567890123456789211";	// 自由使用欄、再発行区分
barCodeData += "020331";			// 支払い期限日
barCodeData += "0";				// 印紙フラグ
barCodeData += "123456";			// 支払い金額
barCodeData += "2";				// 全体チェックデジット

// バーコードの種類を「GS1Fixed」に設定します。
barCode.setType(BarCodeType.GS1Fixed);
// バーコード生成単位を「Pixel」に設定します。
barCode.setUnit(Unit.Pixel);
// バーコードの横方向の解像度を「600」に設定します。
barCode.setDpiX(600);
// バーコードの縦方向の解像度を「600」に設定します。
barCode.setDpiY(600);
// バーコードの最小エレメント値を「6」に設定します。
barCode.setElement(6);
// バーコードデータを設定します。
barCode.setValue(barCodeData);
// バーコードイメージを保存します。
try {
  barCode.saveImage("c:\\barcode.png", "PNG");
} catch (IOException e) {
  e.printStackTrace();
}

参考 : 全体チェックデジットの計算方法

全体チェックデジットは、「モジュラス 10 / ウェイト1、3」で算出します。

  1. 全体デジットの桁を 1 桁目とし、右の桁から左の桁へ順に番号を付加します。
  2. A : 全ての偶数桁の数字を加算して、それに 3 を掛けます。
  3. B : 全体チェックデジットの桁 (1 桁目) を除く、全ての奇数桁の数字を加算します。
  4. C : A と B を加算して、合計を求めます。
  5. 10 から C の下 1 桁の数字を引いたものが全体チェックデジットになります。

※ "9191234512345678901234567892110203310123456" の場合
