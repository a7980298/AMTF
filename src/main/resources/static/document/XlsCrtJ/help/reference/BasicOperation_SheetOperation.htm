ExcelCreator for Java
シートを追加、コピー、削除する
参照

ExcelCreator for Java では、Excel ファイルのシートを追加、コピー、削除することが可能です。このトピックでは、シートを追加、コピー、削除する方法を説明します。

このトピックは、次のセクションを含みます。

  • シートの追加
  • シートのコピー
  • シートの削除

シートの追加

このセクションでは、ブックにシートを追加する方法を説明します。追加するシート数とシートを追加する位置(0～)を指定することができます。

次のコードは、addSheet メソッドを使用して 1 シート目から 2 シート分追加するサンプルコードです。

コピーサンプルコード
// 1 シート目から 2 シート分追加します。
xlsxCreator.addSheet(0, 2);

シートのコピー

このセクションでは、シートをコピーする方法を説明します。コピーするシートの番号(0～)、コピー先のシートの位置(0～)、コピー先のシート名を指定することができます。

次のコードは、copySheet メソッドを使用して 1 シート目をコピーし、シート名を "コピー先シート" に設定して 2 シート目に挿入するサンプルコードです。

コピーサンプルコード
// 1 シート目をコピーし、シート名を "コピー先シート" に設定して 2 シート目に挿入します。
xlsxCreator.copySheet(0, 1, "コピー先シート");

シートの削除

このセクションでは、シートを削除する方法を説明します。削除するシートの番号(0～)、削除するシート数を指定することができます。

次のコードは、deleteSheet メソッドで 1 シート目から 2 シート分削除するサンプルコードです。

コピーサンプルコード
// 1 シート目から 2 シート分削除します。
xlsxCreator.deleteSheet(0, 2);
