ExcelCreator for Java
ExcelCreator for Java の対応する Excel 修飾機能
参照

ExcelCreator for Java で設定可能な Excel の機能および、 既存の Excel ファイルに設定してある Excel の各種設定が ExcelCreator for Java の Excel、PDF ファイル出力時に反映されるかを表します。掲載のない機能は、基本的に ExcelCreator for Java で動作保証を行っていない機能となりますが、詳細については、サポートサービスよりお問い合わせください。

項目について

新規作成 ExcelCreator for Java で Excel ファイルの作成と編集、PDF 出力を行う場合に対応する Excel の機能を表します。
オーバーレイ 既存の Excel ファイルを使用してオーバーレイオープンを行う場合に、オーバーレイ元ファイルに設定してある Excel の各種設定が ExcelCreator for Java の Excel、PDF ファイル出力時に引き継ぐことができるかを表します。

メモ:

(*1)Excel シートをまたいだ処理には未対応。

(*2)同一 Excel ファイルのシートコピーのみ対応。

(*3)新規に Excel ファイルを作成する場合のみシート数を設定することができます。

(*4)ブック名が設定された名前の定義には未対応。

(*5)同一セル内、テキストボックス内で複数の書式を設定することはできません。

(*6)グラフを編集することはできません。

(*7)PDF ファイルの出力には未対応。

(*8)PDF ファイルの出力は [横位置]-[繰り返し] に未対応。

(*9)PDF ファイルに描画可能なオートシェイプは(右図)のみです。

(*10)条件付き書式はコード上からの設定、編集、コピー アンド ペーストには未対応。
