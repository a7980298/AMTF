Web サンプルを使用する
参照

このトピックでは、Web アプリケーション用サンプルの開発環境ごとの使用方法を説明します。

このトピックは、次のセクションを含みます。

  • Eclipse で使用する
  • NetBeans で使用する
メモ:

ExcelCreator for Java の JAR ファイルは、製品インストールフォルダ内 bin フォルダに格納されます。

メモ:

<製品インストールフォルダ>\samples\web\standard サンプルを使用した場合の手順になります。

Eclipse で使用する

  1. Eclipse を起動し、[ファイル] - [新規] - [動的 Web プロジェクト] を選択します。

  2. 任意のプロジェクト名を入力します。

  3. ターゲット・ランタイムは、"Tomcat(Java7)"以降のバージョンを指定し、[完了] をクリックしてプロジェクトを作成します。

  4. 以下のフォルダから [src]、[WebContent] フォルダ一式をコピーします。

    <製品インストールフォルダ>\samples\web\standard\

  5. Eclipse 画面にてパッケージ・エクスプローラー内から作成したプロジェクトを選択し、右クリック後 [貼り付け] を選択します。

  6. プロジェクト名を右クリックして [プロパティー] を開き、[Java のビルド・パス] を選択します。

  7. [ライブラリー] タブを選択し、[外部 JAR の追加] をクリックします。

  8. 製品インストールフォルダ内の JAR ファイルを選択して追加します。

  9. 製品インストールフォルダ内の JAR ファイルをコピーします。

  10. Eclipse 画面にてパッケージ・エクスプローラー内から作成したプロジェクトを選択してツリーを展開し、[WebContent] - [WEB-INF] 内の [lib] を右クリック後 [貼り付け] を選択します。

  11. プロジェクトを選択し [実行] - [サーバーで実行] から実行します。

NetBeans で使用する

  1. NetBeans を起動し、[ファイル] - [新規プロジェクト] - [Java Web] - [Web アプリケーション] を選択し [次へ] をクリックします。

  2. 任意のプロジェクト名を入力します。

  3. [ライブラリの格納用に専用フォルダを使用] にチェック ON (既定 .\lib) 後、[次へ] をクリックします。

  4. サーバーと Java EE バージョン(推奨 Java EE 7 以降)を選択後 [終了] をクリックしてプロジェクト作成します。

  5. NetBeans 画面にてプロジェクトウィンドウ内から作成したプロジェクトを選択後、ツリーを展開し、"index.jsp" を削除します。

  6. 以下のフォルダ内の [src] フォルダ内のファイル一式をコピーします。

    <製品インストールフォルダ>\samples\web\standard\src

  7. NetBeans 画面にてプロジェクトウィンドウ内から作成したプロジェクトを選択後、ツリーを展開し、[ソース・パッケージ] を右クリック後 [貼付け] を選択します。

  8. 以下のフォルダ内の [WebContent] フォルダ内のファイル一式をコピーします。

    <製品インストールフォルダ>\samples\web\standard\WebContent

  9. NetBeans 画面にてプロジェクトウィンドウ内から作成したプロジェクトを選択後、ツリーを展開し、[Web ページ] を右クリックして [貼付け] を選択します。

  10. プロジェクト名を右クリックして [プロパティ] を開き、[ライブラリ] を選択します。

  11. [コンパイル] タブを選択し [JAR/フォルダの追加] をクリックします。

  12. 製品インストールフォルダ内の JAR ファイルを選択して追加し、[OK] でプロパティー画面を閉じます。

  13. プロジェクトを選択し [実行] します。
