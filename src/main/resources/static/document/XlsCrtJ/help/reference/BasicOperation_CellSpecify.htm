セルを指定する

ExcelCreator for Java では、getCell / getPos メソッドを使用し、さまざまな方法でセルを指定して値を設定することが可能です。このトピックでは、セルの指定方法を説明します。

このトピックは、次のセクションを含みます。

  • A1 参照形式によるセルの指定
  • 座標によるセルの指定
  • 変数名によるセルの指定
  • セル名によるセルの指定

A1 参照形式によるセルの指定

このセクションでは、A1 参照形式でセルを指定する方法を説明します。

次のコードは、getCell メソッドを使用して A1 参照形式で A1 セルを指定し、文字列 "アドバンスソフトウェア" を設定するサンプルコードです。

コピーサンプルコード
// A1 セルに文字列 "アドバンスソフトウェア" を設定します。
xlsxCreator.getCell("A1").setValue("アドバンスソフトウェア");

座標によるセルの指定

このセクションでは、座標形式でセルを指定する方法を説明します。

メモ:

座標形式で指定する場合、getPos メソッドの第一引数が x 軸(列)、第二引数が y軸(行) になり、A1 セルを (0, 0) とします。

次のコードは、getPos メソッドを使用して座標形式で A10 セルを指定し、文字列 "アドバンスソフトウェア" を設定するサンプルコードです。

コピーサンプルコード
// A10 セルに文字列 "アドバンスソフトウェア" を設定します。
xlsxCreator.getPos(0, 9).setValue("アドバンスソフトウェア");

変数名によるセルの指定

ExcelCreator for Java では、Excel ファイルのセルに予め設定された、連続した 2つのアスタリスク(*)から始まる文字列を変数名として扱います。このセクションでは、変数名でセルを指定する方法を説明します。

メモ:
既定では、連続した 2つのアスタリスク (*) から始まる文字列を変数名として扱いますが、setKeyword メソッドを使用することで、変数名のキーワード文字をアスタリスク (*) から別の文字に変更することが可能です。

次のコードは、getCell メソッドで A1 セルに設定した変数名 "**Name" を指定し、文字列 "アドバンスソフトウェア" を設定するサンプルコードです。

コピーサンプルコード
// A1 セルに予め変数名 "**Name" を設定し、getCell メソッドで変数名を指定して、文字列 "アドバンスソフトウェア" を設定します。
xlsxCreator.getCell("**Name").setValue("アドバンスソフトウェア");

セル名によるセルの指定

ExcelCreator for Java では、Excel ファイルのセルに予め設定されたセル名を指定することができます。このセクションでは、セル名でセルを指定する方法を説明します。

メモ:
A1 から B2 セルなど、複数セルに設定されたセル名を指定した場合、一番左上のセル (A1 セル) のみが指定されます。

次のコードは、getCell メソッドで A1 セルに設定したセル名 "CELLNAME" を指定し、文字列 "アドバンスソフトウェア" を設定するサンプルコードです。

コピーサンプルコード
// A1 セルに予めセル名 "CELLNAME" を設定し、getCell メソッドでセル名を指定して、文字列 "アドバンスソフトウェア" を設定します。
xlsxCreator.getCell("CELLNAME").setValue("アドバンスソフトウェア");
