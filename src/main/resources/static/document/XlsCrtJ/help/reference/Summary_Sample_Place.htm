ExcelCreator for Java には、クライアント用、Web 用のサンプルファイルが付属しています。

サンプルファイルの既定の場所を以下の表に示します。 %install% は、ExcelCreator for Java のインストールフォルダ (C:\Program Files\XlsCrtJ\ など) を表します。

サンプルファイルの種類サンプルファイルの場所
Java 機能紹介サンプル (クライアント)\%install%\samples\client
Java 機能紹介サンプル (Web)\%install%\samples\web
