サーバーアプリケーション

ExcelCreator for Java を使用して作成したアプリケーションを運用するサーバーに対し、サーバーライセンスが必要です。

ExcelCreator for Java サーバーライセンス

必要条件 ExcelCreator for Java を使用して作成したサーバーアプリケーション(関連する機能を含む)を運用するサーバー環境(OS)数分、必要なライセンスです。
価格 1 ライセンスあたり / 120,000円 (税込 129,600円)
商品構成ライセンス数や契約内容を記載した契約書(ライセンス証書)のみになります。
保守 保守契約が必要となる場合、サーバーライセンス保守契約(年間)の購入が必要になります。
