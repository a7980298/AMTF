ExcelCreator for Java では、getCell / getPos メソッドでセルを指定し、 setValue メソッドで目的のデータを設定することが可能です。このトピックでは、値の設定方法を説明します。

値を設定する

次のコードは、それぞれのセルに文字列、数値、日付を設定するサンプルコードです。

メモ:
日付を設定する場合、予めテンプレートの Excel ファイルに日付の表示形式を設定しておく必要があります。
コピーサンプルコード
// A1 セルに文字列 "Advance Software" を設定します。
xlsxCreator.getCell("A1").setValue("Advance Software");
// A2 セルに整数値 100 を設定します。
xlsxCreator.getCell("A2").setValue(100);
// A3 セルに小数値 1.5 を設定します。
xlsxCreator.getCell("A3").setValue(1.5);
// 現在時刻のシリアル値を取得して setValue メソッドで A4 セルに設定します。
Date date = new Date();
// 86400000d = 1 日のミリ秒数(60s * 60m * 24h * 1000ms)
// 25569d = Java と Excel の基準日の違いによる日数差
// 0.375d = Java(GMT) と Excel(JST) の標準時の仕様の差異による時刻差(9 時間)
double time = date.getTime() / 86400000d + 25569d + 0.375d;
xlsxCreator.getCell("A4").setValue(time);
