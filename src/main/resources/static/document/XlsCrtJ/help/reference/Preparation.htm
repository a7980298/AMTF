JAR ファイルを追加する

このトピックでは、開発環境ごとの ExcelCreator for Java が提供する JAR ファイルの追加方法を説明します。

このトピックは、次のセクションを含みます。

  • Eclipse で開発する
  • NetBeans で開発する
メモ:

ExcelCreator for Java の JAR ファイルは、製品インストールフォルダ内 bin フォルダに格納されます。

Eclipse で開発する

このセクションでは、Eclipse で ExcelCreaotr for Java の JAR ファイルをプロジェクトの新規作成時に追加する方法を説明します。

クライアントアプリケーションで ExcelCreator for Java の JAR ファイルを追加するには

  1. プロジェクトを右クリックし、[プロパティー] をクリックします。

  2. プロパティーウィンドウ内の [Java のビルド・パス] をクリックし、[ライブラリー] タブをクリックします。

  3. [外部 JAR の追加] をクリックし、ExcelCreator for Java の JAR ファイルを選択して [開く] をクリックします。

  4. [ビルド・パス上の JAR およびクラス・フォルダー] に追加されていることを確認し、[OK] をクリックします。

  5. [パッケージ・エクスプローラー] に追加されます。

Web アプリケーションで ExcelCreator for Java の JAR ファイルを追加するには

  1. プロジェクト内の lib フォルダを右クリックし、[新規] - [ファイル] をクリックします。

  2. [拡張] をクリックし、[ファイル・システム内のファイルにリンク] をチェックします。

  3. [参照] をクリックし、ExcelCreator for Java の JAR ファイルを選択して [完了] をクリックします。

NetBeans で開発する

このセクションでは、NetBeans で ExcelCreaotr for Java の JAR ファイルをプロジェクトに追加する方法を説明します。

クライアントアプリケーションで ExcelCreator for Java の JAR ファイルを追加するには

  1. プロジェクト名を右クリックし、[プロパティ] を開き、[ライブラリ] を選択します。

  2. [コンパイル] タブを選択し、[JAR/フォルダの追加] をクリックします。

  3. ExcelCreator for Java の JAR ファイルを選択して追加します。

Web アプリケーションで ExcelCreator for Java の JAR ファイルを追加するには

  1. プロジェクト名を右クリックし、[プロパティ] を開き、[ライブラリ] を選択します。

  2. [コンパイル] タブを選択し、[JAR/フォルダの追加] をクリックします。

  3. ExcelCreator for Java の JAR ファイルを選択して追加します。
