Excel ファイルを上書きオープンする
参照

ExcelCreator for Java では、openBook メソッドを使用して既存の Excel ファイルを上書きオープンすることが可能です。処理の最後には、closeBook メソッドを呼び出し、Excel ファイルをクローズする必要があります。

Excel ファイルを上書きオープンする

次のコードは C ドライブにある Excel ファイル (Template.xlsx) を上書きでオープンするサンプルコードです。

コピーサンプルコード
// C ドライブにある Excel ファイル (Template.xlsx) を上書きでオープンします。
xlsxCreator.openBook("C:\\Template.xlsx", "");
xlsxCreator.closeBook(true);
