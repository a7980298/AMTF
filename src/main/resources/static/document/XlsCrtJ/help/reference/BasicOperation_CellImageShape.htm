ExcelCreator for Java
セルに画像、オートシェイプを設定する
参照

ExcelCreator for Java では、セルに画像やオートシェイプを設定することが可能です。このトピックでは、画像やオートシェイプを設定する方法を説明します。

このトピックは、次のセクションを含みます。

  • 画像を設定する
  • ラインを設定する
  • オートシェイプを設定する
  • テキストボックスを設定する

画像を設定する

このセクションでは、セルに画像を設定する方法を説明します。

次のコードは、A1 セルに画像 [幅：100px、高さ：100px] を設定するサンプルコードです。

コピーサンプルコード
// スケールモードを [ピクセル(px)] に設定します。
xlsxCreator.setScaleMode(ScaleMode.Pixel);
// A1 セルに画像 [幅：100px、高さ：100px] を設定します。
xlsxCreator.getCell("A1").getDrawing().addImage("c:\\image.bmp", 100, 100);

ラインを設定する

このセクションでは、セルにラインを設定する方法を説明します。

次のコードは、A1 セルに [直線 (幅：100px、高さ：100px)] を設定するサンプルコードです。

コピーサンプルコード
// スケールモードを [ピクセル(px)] に設定します。
xlsxCreator.setScaleMode(ScaleMode.Pixel);
// A1 セルに直線 [幅：100px、高さ：100px] を設定します。
xlsxCreator.getCell("A1").getDrawing().addLine(ShapeType.Line, 100, 100);

オートシェイプを設定する

このセクションでは、セルにオートシェイプを設定する方法を説明します。

次のコードは、A1 セルにオートシェイプ [円/楕円 (幅：100px、高さ：100px)] を設定するサンプルコードです。

コピーサンプルコード
// スケールモードを [ピクセル(px)] に設定します。
xlsxCreator.setScaleMode(ScaleMode.Pixel);
// A1 セルにオートシェイプ [円/楕円 幅：100px、高さ：100px] を設定します。
xlsxCreator.getCell("A1").getDrawing().addShape(ShapeType.Ellipse, 100, 100);

テキストボックスを設定する

このセクションでは、セルにテキストボックスを設定する方法を説明します。

次のコードは、A1 セルにテキストボックス [円/楕円 (幅：100px、高さ：100px)] を設定し、ボックス内に文字列 [アドバンスソフトウェア] を設定するサンプルコードです。

コピーサンプルコード
// スケールモードを [ピクセル(px)] に設定します。
xlsxCreator.setScaleMode(ScaleMode.Pixel);
// A1 セルにテキストボックス [円/楕円 幅：100px、高さ：100px] を設定し、ボックス内に文字列 [アドバンスソフトウェア] を設定します。
xlsxCreator.getCell("A1").getDrawing().addTextBox(ShapeType.Ellipse, "アドバンスソフトウェア", 100, 100);
