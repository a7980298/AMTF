セルの属性を設定する
参照

ExcelCreator for Java では、セルの属性を設定することが可能です。このトピックでは、属性を設定する方法を説明します。

このトピックは、次のセクションを含みます。

  • フォント属性を設定する
  • 書式を設定する
  • セルを結合する
  • 文字列の縦位置、横位置を設定する
  • セルの背景色、パターンを設定する
  • セルの罫線を設定する

フォント属性を設定する

このセクションでは、フォントの属性を設定する方法を説明します。

次のコードは、A1 セルのフォント属性 [ＭＳ 明朝、太字、14pt、赤字] を設定するサンプルコードです。

コピーサンプルコード
// フォント名 [ＭＳ 明朝] を設定します。
xlsxCreator.getCell("A1").getAttr().setFontName("ＭＳ 明朝");
// フォントカラー [赤] を設定します。
xlsxCreator.getCell("A1").getAttr().setFontColor(ExcelColor.Red);
// フォントスタイル [太字] を設定します。
xlsxCreator.getCell("A1").getAttr().setFontStyle(FontStyle.Bold);
// フォントポイント [14] を設定します。
xlsxCreator.getCell("A1").getAttr().setFontPoint(14.0);

書式を設定する

このセクションでは、書式を設定する方法を説明します。

次のコードは、setFormat メソッドで A1 セルに数値の書式と数値 1,000,000 を設定するサンプルコードです。

コピーサンプルコード
// A1 セルに数値の書式を設定します。
xlsxCreator.getCell("A1").getAttr().setFormat("#,##0");
// A1 セルに数値 1,000,000 を設定します。
xlsxCreator.getCell("A1").setLong(1000000);

セルを結合する

このセクションでは、セルを結合する方法を説明します。

次のコードは、setMergeCells メソッドで A1:C3 セルを結合するサンプルコードです。

コピーサンプルコード
// A1:C3 セルを結合します。
xlsxCreator.getCell("A1:C3").getAttr().setMergeCells(true);

文字列の縦位置、横位置を設定する

このセクションでは、文字列の縦位置、横位置を設定する方法を説明します。

次のコードは、setVerticalAlignment メソッドで A1 セルの文字列の縦位置 [上詰め]、setHorizontalAlignment メソッドで B1 セルの文字列の横位置 [右詰め] を設定するサンプルコードです。

コピーサンプルコード
// A1 セルに文字列の縦位置 [上詰め] を設定します。
xlsxCreator.getCell("A1").getAttr().setVerticalAlignment(VerticalAlignment.Top);
// B1 セルに文字列の横位置 [右詰め] を設定します。
xlsxCreator.getCell("B1").getAttr().setHorizontalAlignment(HorizontalAlignment.Right);

セルの背景色、パターンを設定する。

このセクションでは、セルの背景色、パターンを設定する方法を説明します。

セルの背景色を設定する

次のコードは、setBackColor メソッドで A1 セルの背景色を [赤] に設定するサンプルコードです。

コピーサンプルコード
// A1 セルに背景色 [赤] を設定します。
xlsxCreator.getCell("A1").getAttr().setBackColor(ExcelColor.Red);

セルのパターンを設定する

次のコードは、setPattern メソッドで A1 セルにパターン [左下がり斜線 縞]、setBackColor メソッドでパターン背景色 [緑]、setForeColor メソッドでパターン前景色 [青] で設定するサンプルコードです。

コピーサンプルコード
// A1 セルにパターン [左下がり斜線 縞] を設定します。
xlsxCreator.getCell("A1").getAttr().setPattern(Pattern.DarkUp);
// A1 セルに背景色 [緑] を設定します。
xlsxCreator.getCell("A1").getAttr().setBackColor(ExcelColor.Green);
// A1 セルに前景色 [青] を設定します。
xlsxCreator.getCell("A1").getAttr().setForeColor(ExcelColor.Blue);

セルの罫線を設定する

このセクションでは、セルの罫線を設定する方法を説明します。

次のコードは、それぞれのセルに上罫線、下罫線、左罫線、右罫線、左上がり罫線、右上がり罫線を設定するサンプルコードです。

コピーサンプルコード
// lineTop メソッドで B2 セルに上罫線（実線、青）を設定します。
xlsxCreator.getCell("B2").getAttr().lineTop(BorderStyle.Thin, ExcelColor.Blue);
// lineBottom メソッドで B3 セルに下罫線（実線、ピンク）を設定します。
xlsxCreator.getCell("B3").getAttr().lineBottom(BorderStyle.Thin, ExcelColor.Pink);
// lineLeft メソッドで C2 セルに左罫線（二重線、黒）を設定します。
xlsxCreator.getCell("C2").getAttr().lineLeft(BorderStyle.Double , ExcelColor.Black);
// lineRight メソッドで C3 セルに右罫線（二重線、緑）を設定します。
xlsxCreator.getCell("C3").getAttr().lineRight(BorderStyle.Double, ExcelColor.Green);
// lineLeftUp メソッドで D2 セルに左上がり罫線（太線、赤）を設定します。
xlsxCreator.getCell("D2").getAttr().lineLeftUp(BorderStyle.Medium , ExcelColor.Red);
// lineRightUp メソッドで D3 セルに右上がり罫線（太線、黄）を設定します。
xlsxCreator.getCell("D3").getAttr().lineRightUp(BorderStyle.Medium, ExcelColor.Yellow);
