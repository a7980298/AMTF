行を挿入、コピー、クリア、削除する
参照

ExcelCreator for Java では、シートの行を挿入、コピー、クリア、削除することが可能です。このトピックでは、行を挿入、コピー、クリア、削除する方法を説明します。

このトピックは、次のセクションを含みます。

  • 行の挿入
  • 行のコピー、貼り付け
  • 行のクリア
  • 行の削除

行の挿入

このセクションでは、行を挿入する方法を説明します。挿入する行(0～)と挿入する行数を指定することができます。

次のコードは、rowInsert メソッドを使用して 1 行目に 5 行分挿入するサンプルコードです。

コピーサンプルコード
// 1 行目に 5 行分挿入します。
xlsxCreator.rowInsert(0, 5);

rowInsertCopied メソッドを使用して、コピーした行を指定の行に挿入することができます。
次のコードは、rowInsertCopied メソッドを使用して コピーした行を 2 行目に 5 行分挿入するサンプルコードです。

コピーサンプルコード
// 1 行目をメモリ上にコピーします。
xlsxCreator.rowCopy(0);
// コピーした行を 2 行目に 5 行分挿入します。
xlsxCreator.rowInsertCopied(1, 5);

行のコピー、貼り付け

このセクションでは、行をコピーする方法を説明します。コピーする行(0～)を指定することができます。第 2 引数で貼り付ける行(0～)を指定することもできます。

次のコードは、rowCopy メソッドを使用して 1 行目をコピーし、2 行目に貼り付けるサンプルコードです。

コピーサンプルコード
// 1 行目をコピーして 2 行目に貼り付けます。
xlsxCreator.rowCopy(0, 1);

行のクリア

このセクションでは、行をクリアする方法を説明します。クリアする行の番号(0～)、クリアする行数を指定することができます。

次のコードは、rowClear メソッドで 1 行目から 5 行分クリアするサンプルコードです。

コピーサンプルコード
// 1 行目から 5 行分クリアします。
xlsxCreator.rowClear(0, 5);

行の削除

このセクションでは、行を削除する方法を説明します。削除する行の番号(0～)、削除する行数を指定することができます。

次のコードは、rowDelete メソッドで 1 行目から 5 行分削除するサンプルコードです。

コピーサンプルコード
// 1 行目から 5 行分削除します。
xlsxCreator.rowDelete(0, 5);
