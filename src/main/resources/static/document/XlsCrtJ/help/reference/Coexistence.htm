ExcelCreator シリーズとの互換性

ExcelCreator for Java は、他の ExcelCreator シリーズとの互換性はありません。

また、移行ツールも用意されておりませんので、作成したプロジェクト、アプリケーションを ExcelCreator for Java で使用することはできません。

ExcelCreator シリーズとの共存

開発環境の共存

ExcelCreator for Java は、他の ExcelCreator シリーズと同一の開発環境に共存させることができます。

実行環境の共存

ExcelCreator for Java は、他の ExcelCreator シリーズと同一の実行環境に共存させることができます。
