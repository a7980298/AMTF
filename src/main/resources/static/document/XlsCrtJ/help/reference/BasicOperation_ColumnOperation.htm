ExcelCreator for Java
列を挿入、コピー、クリア、削除する
参照

ExcelCreator for Java では、シートの列を挿入、コピー、クリア、削除することが可能です。このトピックでは、列を挿入、コピー、クリア、削除する方法を説明します。

このトピックは、次のセクションを含みます。

  • 列の挿入
  • 列のコピー、貼り付け
  • 列のクリア
  • 列の削除

列の挿入

このセクションでは、列を挿入する方法を説明します。挿入する列(0～)と挿入する列数を指定することができます。

次のコードは、columnInsert メソッドを使用して 1 列目に 5 列分挿入するサンプルコードです。

コピーサンプルコード
// 1 列目に 5 列分挿入します。
xlsxCreator.columnInsert(0, 5);

columnInsertCopied メソッドを使用して、コピーした列を指定の列に挿入することができます。
次のコードは、columnInsertCopied メソッドを使用して コピーした列を 2 列目に 5 列分挿入するサンプルコードです。

コピーサンプルコード
// 1 列目をメモリ上にコピーします。
xlsxCreator.columnCopy(0);
// コピーした列を 2 列目に 5 列分挿入します。
xlsxCreator.columnInsertCopied(1, 5);

列のコピー、貼り付け

このセクションでは、列をコピーする方法を説明します。コピーする列(0～)を指定することができます。第 2 引数で貼り付ける列(0～)を指定することもできます。

次のコードは、columnCopy メソッドを使用して 1 列目をコピーし、2 列目に貼り付けるサンプルコードです。

コピーサンプルコード
// 1 列目をコピーして 2 列目に貼り付けます。
xlsxCreator.columnCopy(0, 1);

列のクリア

このセクションでは、列をクリアする方法を説明します。クリアする列の番号(0～)、クリアする列数を指定することができます。

次のコードは、columnClear メソッドで 1 列目から 5 列分クリアするサンプルコードです。

コピーサンプルコード
// 1 列目から 5 列分クリアします。
xlsxCreator.columnClear(0, 5);

列の削除

このセクションでは、列を削除する方法を説明します。削除する列の番号(0～)、削除する列数を指定することができます。

次のコードは、columnDelete メソッドで 1 列目から 5 列分削除するサンプルコードです。

コピーサンプルコード
// 1 列目から 5 列分削除します。
xlsxCreator.columnDelete(0, 5);
