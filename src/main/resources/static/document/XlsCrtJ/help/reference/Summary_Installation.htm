製品のインストール

ExcelCreator for Java は、インストールフォルダが zip 形式で圧縮されています。ファイルをローカルにコピー後、インストールフォルダを解凍してご対応ください。

JAR ファイル名
excelcreator-1.1.0.XXXX.jar
excelcreator-1.1.jar
barcodecreator-1.0.0.XXXX.jar
barcodecreator-1.0.jar
excelcreator.presetfont.bin
メモ:

JAR ファイルは、 ExcelCreator for Java インストールフォルダ内の bin フォルダに格納されます。

製品のアンインストール

本製品をアンインストールする場合、ローカルにコピーした解凍後のインストールフォルダを削除してご対応ください。
