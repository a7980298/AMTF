ExcelCreator for Java を使用して作成したアプリケーションを配布する場合、以下の再配布可能ファイルをアプリケーションと一緒に配布してください。

このトピックは、次のセクションを含みます。

  • 再配布可能ファイル
  • 再配布可能ファイルの格納場所
  • 再配布可能ファイルの配置
  • 再配布の条件

再配布可能ファイル

ファイル名内容
excelcreator-1.1.0.XXXX.jar
excelcreator-1.1.jar		ExcelCreator for Java JAR ファイル
barcodecreator-1.0.0.XXXX.jar
barcodecreator-1.0.jar		ExcelCreator for Java バーコード生成 JAR ファイル
excelcreator.presetfont.binプリセット フォント ファイル

再配布可能ファイルの格納場所

再配布可能ファイル (JAR) は、製品解凍先の下記のフォルダに格納されます。

<解凍先フォルダ>\bin

再配布可能ファイルの配置

再配布可能ファイル (JAR) は、以下のいずれかの方法で実行環境に配布します。

  • アプリケーションに取り込み配布する
  • 共有フォルダへ配置する
