PDF ファイルを出力する

次のコードは、C ドライブにある Excel ファイル (Template.xlsx) をオーバーレイオープンして C ドライブに Out.xlsx と Out.pdf を出力するサンプルコードです。
第 3 引数に HTML ファイル名を指定することで、HTML ファイルを出力することもできます。

コピーサンプルコード
// C ドライブにある Excel ファイル (Template.xlsx) をオーバーレイオープンして C ドライブに Out.xlsx と Out.pdf を出力します。
xlsxCreator.openBook("C:\\Out.xlsx", "C:\\Template.xlsx");
// PDF ファイルの文書のプロパティ「タイトル」に "ExcelCreator for Java PDF Title" を設定します。
xlsxCreator.getPdf().setTitle("ExcelCreator for Java PDF Title");
// PDF ファイルの文書のプロパティ「サブタイトル」に "ExcelCreator for Java PDF Subject" を設定します。
xlsxCreator.getPdf().setSubject("ExcelCreator for Java PDF Subject");
// PDF ファイルの文書のプロパティ「作成者」に "Advance Software PDF Author" を設定します。
xlsxCreator.getPdf().setAuthor("Advance Software PDF Author");
// PDF ファイルの文書のプロパティ「PDF 変換」に "Advance Software PDF Producer" を設定します。
xlsxCreator.getPdf().setProducer("Advance Software PDF Producer");
// C ドライブに Out.pdf を出力します。
xlsxCreator.closeBook(true, "C:\\Out.pdf", "", false);

フォントについて

PDF ファイル出力を行う環境では、予め Excel ブック内で指定されるフォント(ファイル)を配布する必要があります。また、「ＭＳ Ｐゴシック」などの他環境への配布が困難な Windows OS や Office 製品のバンドルフォントについては、「プリセットフォント機能」により制限付きでの文字出力に対応します。

フォントディレクトリ

Windows OS、RHEL、CentOS 既定のフォントディレクトリは下記の通りです。但し、フォントディレクトリは OS のインストール方法やディストリビューション、バージョンで異なる場合があります。

  • 各 OS のフォントディレクトリ
    • Windows OS "C:\Windows\Fonts\"
    RHEL, CentOS "~/.fonts/" または "/usr/share/fonts/"

    フォントファイル、外字ファイルの配布時の注意点

    実行環境にフォントファイルを配置する場合、次の点にご注意ください。

  • 外字(EUDC.TTE)ファイルはフォントディレクトリに配置するか、getPdf.setEudcFiles メソッドでの指定が必要です。
  • ttf 、ttc 形式のフォントファイルに対応します。
  • フォントの配布時は、著作者に対し配布や利用条件を確認いただく必要があります。また、弊社では配布に関する内容にお答えすることはできません。

    • プリセットフォント機能

    プリセットフォントは、Windows OS や Office 製品のバンドルフォントの配布なしで文字出力を実現できる機能です。次のフォントは設定なしで機能します。

    ＭＳ Ｐゴシック, ＭＳ ゴシック, ＭＳ Ｐ明朝, ＭＳ 明朝, メイリオ, 游明朝, 游ゴシック, Arial, Times New Roman, Tahoma, Microsoft Sans Serif

    また、上記に該当しないバンドルフォントは、外部プリセットファイル(ファイル名：excelcreator.presetfont.bin)を製品 Jar ファイルの同一ディレクトリに配置することで機能します。

    プリセットフォント機能の動作

    プリセットフォント参照時は次の仕様で出力されます。

  • 文字集合規格「Adobe-Japan1-2」に収録される文字の出力に対応します。
  • 上記の文字集合に該当しない文字は '?' で出力されます。
  • 文字の埋め込みは行われません。

    • 該当フォントが無い場合の動作

    プリセットフォントを含め該当フォントがない場合は、次の仕様で出力されます。

  • 製品内部のフォントリンク情報を参照し、他のフォントでの出力を試みます。
  • 該当フォントが存在しない場合、文字種により次の文字で出力されます。
        全角文字・・・「Microsoft Sans Serif」で文字'?'が出力されます。
        半角文字・・・「Microsoft Sans Serif」で指定文字が出力されます。
  • 文字の埋め込みは行われません。
