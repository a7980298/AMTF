セルにバーコードを設定する

ExcelCreator for Java では、付属の BarCodeCreator を使用して作成したバーコードイメージを指定したセルに設定することが可能です。このトピックでは、バーコードイメージの作成とセルへの設定方法を説明します。

セルにバーコードを設定する

次のコードは、BarCodeCreator を使用して作成したバーコードイメージを指定したセルに設定するサンプルコードです。

コピーサンプルコード
// BarCodeCreator のインスタンスを生成します。
BarCodeCreator barCode = new BarCodeCreator();
// バーコードの種類に「JAN13」を設定します。
barCode.setType(BarCodeType.JAN13);
// バーコードの生成単位に「Pixel」を設定します。
barCode.setUnit(Unit.Pixel);
// バーコードの X 方向の解像度に「600」を設定します。
barCode.setDpiX(600);
// バーコードの Y 方向の解像度に「600」を設定します。
barCode.setDpiY(600);
// バーコードの値に「1234567890123」を設定します。
barCode.setValue("1234567890123");
// Millimeter 単位でバーコードのサイズを取得します。
BarCodeSizeD size = barCode.getScaleSize(Unit.Millimeter);
// B2 セルに作成したバーコードイメージを取得したサイズで設定します。
xlsxCreator.getCell("B2").getDrawing().addImage(barCode.getImage(), size.getWidth(), size.getHeight());
