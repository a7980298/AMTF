ExcelCreator for Java 体験版は、製品版と同様に全ての機能をお試しいただくことが可能ですが、以下の制限事項がございます。

制限事項

  • Excel ファイルを作成すると、作成された全てのシートのフッターに体験版であることをあらわすメッセージ文字列が出力されます。
  • バーコードクラスで作成したバーコードイメージの左上に「AdvanceSoftware」の文字列が出力されます。
  • 体験版の JAR ファイルは実行環境での動作検証以外の目的で配布することはできません。

注意:

製品版をインストールする場合、体験版を必ずアンインストールしてからインストールを実行してください。
