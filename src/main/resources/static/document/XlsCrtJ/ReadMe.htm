■ はじめに

ExcelCreator for Java は、Java EE、Java SE 環境で作成したプログラム中から Excel ファイルを高速に生成することができるコンポーネントです。独自の技術でファイルの生成を行うため処理速度が非常に高速です。Excel ファイルの新規作成、既存の Excel ファイルの上書きや別ファイルへの保存(オーバーレイ)が可能で、目的のセルに対してデータの設定の他、セルの書式、計算式、画像、オートシェイプ等の設定、シートのコピーや行、列の挿入/削除等がシンプルなコーディングで実現できます。また、PDF ファイルや HTML ファイルを出力することもできます。

ExcelCreator for Java 製品版ユーザー様は、弊社 Web サイト よりユーザー登録の手続きを行ってください。ユーザー登録が完了次第、サポートサービスのご利用が可能になります。

ファイルの説明

上記のフォルダ、ファイル構成で主要なものを説明します。

  • フォルダ／ファイル名 説明

■ インストールとアンインストール

本製品のインストールは、zip 形式の圧縮ファイル(ExcelCreatorJavaSetup_XXXX.zip)を解凍後、ローカルの任意の箇所にコピーしてご対応ください。アンインストールは、解凍後のインストールフォルダ(XlsCrtJ)一式を削除してください。
